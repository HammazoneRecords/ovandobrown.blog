# **App Name**: Orbital Narratives

## Core Features:

- Homepage Immersion: Non-linear homepage design featuring 3D imagery for an immersive experience.
- Post Categories: Categorized posts (linked to ovandobrown.blog) for seamless navigation.
- Interactive Comments: Interactive comments section for each post.
- About Ovando: Detailed 'About Ovando' page.
- Site Search: Site-wide search function.
- Dark Mode: Toggle to enable a sophisticated dark mode.
- Related Content Suggestions: AI-powered tool that suggests relevant past posts or comments when a user is viewing a specific article.

## Style Guidelines:

- Background color: Dark gray (#121212) to establish a sophisticated dark mode.
- Primary color: Emerald Purple (#6B3FA0) for interactive elements to infuse elegance and futurism.
- Accent color: Deep Blue (#304FFE) used sparingly to draw attention to calls to action without overpowering the dark mode.
- Body font: 'Inter', a sans-serif font with a modern look, well-suited to body text. Headline font: 'Space Grotesk', also sans-serif; it can be used for headlines.
- Minimalist icons that are geometric and futuristic, in a light emerald purple. Only a small number of icons should be used.
- Non-linear, asymmetrical layout with staggered content blocks to enhance the futuristic and exploratory feel. Use 3D image assets placed artfully across the page, ensuring text is still readable.
- Subtle parallax scrolling effects with 3D imagery to create depth and a sense of movement. Loading animations are clean and smooth to match a premium design.
